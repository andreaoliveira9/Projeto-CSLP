\doxysubsection{Detailed Description}
Encodes video using hybrid coding techniques. 

The \doxylink{class_hybrid_encoder}{Hybrid\+Encoder} class is designed to encode video using a combination of motion-\/compensated predictive coding (inter-\/coding) and transform coding (intra-\/coding).
