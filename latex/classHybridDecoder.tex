\doxysubsection{Detailed Description}
Class for implementing hybrid video decoding. 

The \mbox{\hyperlink{classHybridDecoder}{Hybrid\+Decoder}} class handles the decoding of video that was encoded using the \mbox{\hyperlink{classHybridEncoder}{Hybrid\+Encoder}} class. It uses motion-\/compensated predictive coding and transform coding to reconstruct video frames from encoded data.
