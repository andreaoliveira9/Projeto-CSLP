\doxysubsection{Detailed Description}
Decodes video that was encoded using hybrid coding techniques. 

The \mbox{\hyperlink{classHybridDecoder}{Hybrid\+Decoder}} class is responsible for decoding video that was encoded by the \mbox{\hyperlink{classHybridEncoder}{Hybrid\+Encoder}} class. It uses motion-\/compensated predictive coding and transform coding to reconstruct the original video frames.
