\doxysubsection{Detailed Description}
Class for implementing hybrid video coding. 

The \mbox{\hyperlink{classHybridEncoder}{Hybrid\+Encoder}} class is designed to encode video using a combination of motion-\/compensated predictive coding and transform coding techniques. It supports efficient encoding of video frames by considering spatial and temporal redundancies.
