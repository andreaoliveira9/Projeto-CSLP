\doxysubsection{Detailed Description}
Encodes video using hybrid coding techniques. 

The \mbox{\hyperlink{classHybridEncoder}{Hybrid\+Encoder}} class is designed to encode video using a combination of motion-\/compensated predictive coding (inter-\/coding) and transform coding (intra-\/coding). It supports efficient encoding by exploiting both spatial and temporal redundancies.
