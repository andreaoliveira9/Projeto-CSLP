\doxysubsection{Detailed Description}
Decodes video that was encoded using hybrid coding techniques. 

The \doxylink{class_hybrid_decoder}{Hybrid\+Decoder} class is responsible for decoding video that was encoded by the \doxylink{class_hybrid_encoder}{Hybrid\+Encoder} class. It uses motion-\/compensated predictive coding and transform coding to reconstruct the original video frames.
